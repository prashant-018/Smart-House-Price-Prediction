# 🏠 Smart House Price Prediction System

An end-to-end Machine Learning project that predicts house prices based on house features using **Linear Regression**, **FastAPI**, and **HTML/CSS/JavaScript**.

---

## 🚀 Features

- 📊 Data Analysis (EDA)
- 🧹 Data Cleaning
- 🔄 Feature Encoding
- 🎯 Feature Selection
- 📈 Feature Scaling
- 🤖 Linear Regression Model
- ⚡ FastAPI REST API
- 🌐 HTML, CSS & JavaScript Frontend
- 💰 Real-Time House Price Prediction

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Smart-House-Price-Prediction.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run FastAPI

```bash
python -m uvicorn app:app --reload
```

### Open Frontend

Open

```
frontend/index.html
```
